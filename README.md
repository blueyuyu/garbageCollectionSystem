## 垃圾分类管理系统-后端

这是一个基于springboot + vue2的垃圾管理系统



## 使用技术

**核心框架**： springboot2  

**Token认证：** jwt

**持久层框架：**mybatis-plus

**构建工具**：maven

**开发基于JDK8**



## 开发工具

**编译器：** idea 2021

**数据库：**Navicat

**接口测试**：Postman
